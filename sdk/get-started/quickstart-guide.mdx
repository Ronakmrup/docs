## Step 1: Install the SDK

1. Download your SDK connector file (e.g., `sanas_remote_sdk_<platform-arch>_<version>.tar.gz`)
2. Extract and install:

```bash
tar -zxvf sanas_remote_sdk_<platform-arch>_<version>.tar.gz
cd sanas_remote_sdk_<platform-arch>_<version>
./install.sh
```

## Verify installation

```bash
python3 -c "import sanas_remote_sdk; print('SDK installed successfully!')"
```

## Step 3: Run Your First Example

Copy and paste this complete working example:

<CodeGroup>

```
import sanas_remote_sdk

# Initialize SDK
sdk = sanas_remote_sdk.CreateRemoteSDK()

init_params = sanas_remote_sdk.InitParams()
init_params.remoteEndpoint = "your_endpoint_url"  # From credentials email
init_params.accountId = "your_account_id"
init_params.accountSecret = "your_account_secret"
init_params.secureMedia = True

result = sdk.Initialize(init_params)
print(f"SDK initialized: {result}")

# State callback
def state_callback(state, reason):
    if state == sanas_remote_sdk.ProcessorState.READY:
        print("Processor ready!")

# Create AI Voice Agent processor
audio_params = sanas_remote_sdk.AudioParams()
audio_params.modelName = "ASR_NC3.0_VAD0.6"
audio_params.sampleRate = 16000  # 16kHz for AI Voice Agent

processor, create_result = sdk.CreateAudioProcessor(audio_params, state_callback)

if create_result == sanas_remote_sdk.CreateProcessorResult.SUCCESS:
    # Process audio chunks (example with dummy data)
    input_audio = [0.1] * 320  # 20ms at 16kHz
    output_audio = processor.ProcessSamples(input_audio)
    print(f"Processed {len(output_audio)} samples")
    
    # Now send output_audio to your ASR system

# Cleanup
sdk.DestroyAudioProcessor(processor)
sdk.Shutdown()
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Import error: No module named 'sanas_remote_sdk'" icon="triangle-exclamation">
    **Cause:** SDK not installed or wrong Python version

    **Fix:**

    ```bash
    python3 --version  # Verify Python 3.10+
    ./install.sh       # Re-run installer
    ```
  </Accordion>
  <Accordion title="Initialize() returns error" icon="triangle-exclamation">
    **Cause:** Wrong credentials or endpoint URL

    **Fix:**

    - Verify `remoteEndpoint`, `accountId`, and `accountSecret` match your credentials email
    - Check network connectivity to Sanas Cloud
    - Try `secureMedia = False` for testing
  </Accordion>
  <Accordion title="CreateAudioProcessor() fails" icon="triangle-exclamation">
    **Cause:** Invalid model name or sample rate

    **Fix:**

    - AI Voice Agent: Use `ASR_NC3.0_VAD0.6` with `sampleRate â‰¤ 16000`
    - Noise Cancellation: Use `ST`, `VI-G`, `VI-GT`, or `VI-N` with appropriate sample rate
  </Accordion>
</AccordionGroup>
