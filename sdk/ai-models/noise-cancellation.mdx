## Introduction

Sanas Noise Cancellation (NC) provides AI-powered noise cancellation that extracts voices from noisy signals in real-time across any acoustic conditions. With low latency and minimal computing cost, NC delivers clean speech waveforms suitable for human communication.

Designed for real-time inference, NC works in both streaming (live audio) and offline (recorded files) scenarios, making it versatile for any voice communication application.

## Model Selection Guide

<Steps>
  <Step title="Identify your scenario">
    **How many speakers do you need to preserve?**

    - Multiple speakers → Use **ST** (Standard)
    - Single speaker → Continue to Step 2
  </Step>
  <Step title="What's your microphone type?">
    **For single speaker scenarios:**

    - Headset/close-proximity mic (~30cm) → Use **VI-N** (Nearfield)
    - Phone/VoIP (8kHz) → Use **VI-GT** (Telephony)
    - General microphone (~1m) → Use **VI-G** (General)
  </Step>
  <Step title="Configure in SDK">
    Set the model name in your audio parameters:

    ```python
    audio_params.modelName = "VI-G"  # Your chosen model
    audio_params.sampleRate = 16000   # Appropriate sample rate
    ```
  </Step>
</Steps>
