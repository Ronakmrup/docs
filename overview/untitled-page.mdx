## Popular Use Cases

<AccordionGroup>
  <Accordion title="Improve Voice Assistant ASR Accuracy" icon="microphone">
    **Challenge:** Your voice agent misunderstands users in noisy environments (cafes, streets, offices).

    **Solution:** Use **AI Voice Agent** model to reduce Word Error Rate (WER) by 5-30%.

    **Quick Start:**

    ```python
    # 30-second example
    from sanas_sdk import create_sdk
    
    sdk = create_sdk(endpoint="...", account_id="...", secret="...")
    processor = sdk.create_processor(model="ASR_NC3.0_VAD0.6")
    
    clean_audio = processor.process(noisy_audio)
    # Pass to your ASR (Google, Deepgram, etc.)
    ```

    [Full Tutorial ‚Üí](/sdk/tutorials/single-stream)
  </Accordion>
