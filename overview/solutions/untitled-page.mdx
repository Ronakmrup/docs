## Overview

Real-world voice calls experience suffers from background noise that degrades audio quality and user experience. Office chatter, traffic sounds, keyboard typing, and ambient noise make conversations difficult and unprofessional.

The **Noise Cancellation** models remove unwanted background noise while preserving natural voice quality, delivering crystal-clear audio in any acoustic environment.

---

## How It Works

<Steps>
  <Step title="Capture audio">
    The microphone captures the speaker's voice + background noise.
  </Step>
  <Step title="Process through Sanas NC">
    Model removes background noise based on the selected variant (multi-speaker, single-speaker, telephony, or headset)
  </Step>
  <Step title="Deliver clean audio">
    Clean audio sent to encoder/network for transmission
  </Step>
  <Step title="Great call experience">
    The recipient hears a clear voice without distractions
  </Step>
</Steps>

## Quick Start (30 seconds)

```python
import sanas_remote_sdk

# Initialize SDK
sdk = sanas_remote_sdk.CreateRemoteSDK()
init_params = sanas_remote_sdk.InitParams()
init_params.remoteEndpoint = "your_endpoint"
init_params.accountId = "your_account_id"
init_params.accountSecret = "your_account_secret"
sdk.Initialize(init_params)

# Create Noise Cancellation processor
audio_params = sanas_remote_sdk.AudioParams()
audio_params.modelName = "VI-G"  # General Purpose
audio_params.sampleRate = 16000

processor, result = sdk.CreateAudioProcessor(audio_params, state_callback)

# Process audio in real-time
for chunk in audio_stream:
    clean_audio = processor.ProcessSamples(chunk)
    # Send clean_audio to encoder/network
```
