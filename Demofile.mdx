## Prerequisites

Ensure the following conditions are met to ensure uninterrupted operations:

- ✅ Requires minimum Python version **3.10**
- ✅ Operating system: Ubuntu **22.04 x86-64**
- ✅ Received SDK access credentials from Sanas. If you haven't, [fill out the access form](https://www.sanas.ai) to request access.

<Info>
  All native libraries are bundled with no external dependencies.
</Info>

# Getting Started

<Steps>
  <Step title="Extract the SDK connector">
    Navigate to the directory where you've downloaded the SDK connector file. Example file name: `sanas_remote_client_sdk_python_1.0.5.tar.gz`

    ```python
    tar -zxvf sanas_remote_client_sdk_python_1.0.5.tar.gz
    cd sanas_remote_client_sdk_python_1.0.5
    ```
  </Step>
  <Step title="Installation">
    Install the required libraries.

    ```python
    python3 -m venv client_sdk
    source client_sdk/bin/activate
    pip3 install sanas_remote_sdk-1.0.5-cp310-cp310-linux_x86_64.whl
    ```

    And that's it, you are all set to start utilizing Sanas Speech AI SDK.
  </Step>
